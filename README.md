A desktop voice assistant built with Python that responds to voice commands and performs common desktop tasks.

Features
🎤 Wake word: "Hey Alex"
🌐 Open Google
▶ Open YouTube
📝 Open Notepad
📂 Open Downloads
🖥 Shutdown confirmation
🖼 Simple Tkinter GUI
🔈 Text-to-Speech responses
Technologies
Python
SpeechRecognition
pyttsx3
Tkinter
PyAudio
webbrowser
subprocess
